Informasi Aplikasi Buku Kas

Aplikasi Buku Kas ini dirancang untuk membantu Anda mencatat, mengelola, dan memantau transaksi keuangan secara efisien dan mudah. Berikut fitur-fitur yang tersedia:

Fitur Utama

  • Manajemen Transaksi: Tambah, edit, hapus transaksi dengan kategori, tanggal, dan nominal yang fleksibel.
  • Manajemen Kategori: Tambah kategori baru dengan input multi-data sekaligus, pilih warna dan ikon untuk tiap kategori.
  • Filter: Filter transaksi berdasarkan tanggal dan kategori.
  • Grafik Interaktif: Menampilkan grafik pemasukan dan pengeluaran per hari.
  • Export: Export data transaksi ke file .csv.
  • Print: Cetak riwayat transaksi lengkap (semua halaman, meskipun data dipaginasi).
  • Backup & Restore: Backup seluruh data dalam format JSON dan import kembali jika diperlukan.
  • Dark Mode: Tersedia tampilan tema gelap untuk kenyamanan mata pengguna.

Fitur Kategori

  • Input Multi: Bisa memasukkan banyak kategori sekaligus dengan memisahkan nama menggunakan koma (,).
  • Ikon & Warna: Setiap kategori dapat memilih ikon dan warna sendiri untuk memudahkan identifikasi saat melihat data.
  • Pembaruan Otomatis: Jika nama kategori sudah ada, maka akan memperbarui data lama berdasarkan nama tersebut, tetap mempertahankan ID lama.

Cara Penggunaan Input Multi Kategori

Saat menambahkan kategori baru:

  • Gunakan tanda koma (,) untuk memisahkan nama kategori, contoh: Transportasi, Makan, Belanja.
  • Gunakan tanda (=>) untuk menyinkronkan ikon dengan kategori dan gunakan tanda koma (,) untuk memisahkan nama kategori contoh: Transportasi => 🚗, Makanan => 🍔, Minuman => 🥛
  • untuk warna kasih sesuai keinginan anda.

Catatan

  • Pastikan selalu melakukan backup berkala untuk menghindari kehilangan data penting.
  • Gunakan fitur Dark Mode untuk kenyamanan penggunaan pada malam hari.
  • Jika Anda mengedit nama kategori, pastikan juga memperbarui data transaksi yang terkait. Misalnya, jika kategori Pakaian diubah menjadi Pakaians, maka data transaksi lama dengan nama "Pakaian" tidak akan tersinkronisasi dengan ikon baru, sehingga ikon pada transaksi tersebut bisa hilang.
  • Setiap kali ada fitur baru yang ditambahkan, pastikan Anda membaca informasi dan panduan terbaru agar tidak melewatkan perubahan penting.
